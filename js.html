Local Storage

This page demonstrages how variables such as background colors and fonts can be
stored in local storage.

This button will reset all local storage values to default.

Change Background Color

Select any color from the drop-down list to change the current page's background color.
This color will also be stored in the local storage.

Change Page Font Families

Select any font type from the drop-down list to change the current page's font type.
This font type will also be stored in the local storage.

Change Page Font Colors

Select any color from the drop-down list to change the current page's font color.
This color will also be stored in the local storage.
